# What is unBackle?
unBackle is a software package to create DB native triggers and functions in order to get rid of backend entirely.
# Why choose unBackle?
Using backend allows you to easily do all that you want but adding abstaction level slows processes, most software engineers have solid SQL knowledge even when using ORMs and in age of AI writing SQL is easier then before.
# What does unBackle provide?
We are in pre-alpha stage.
### GOALS:
1. Provide authentication and authorization layers into SQL that you can safely reuse in your projects.
2. Provide backend-like strong parameters services
3. Provide testing suite for SQL
4. Provide guidance in writing SQL functions for more complex mechanisms
